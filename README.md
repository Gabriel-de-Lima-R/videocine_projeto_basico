# VideoCine

Bem-vindo ao **VideoCine**! Este é um projeto Java desenvolvido com fins educacionais para praticar os conceitos fundamentais da programação orientada a objetos (POO). A ideia central é modelar um catálogo de vídeos, como filmes e séries, utilizando uma hierarquia de classes.

## 📚 Sobre o Projeto

O sistema permite criar e gerenciar objetos do tipo `Titulo`, que servem como classe base para dois tipos específicos de mídia:
- **Filme**: Representa um filme com atributos como duração, ano de lançamento, etc.
- **Serie**: Representa uma série com atributos adicionais como número de temporadas, episódios por temporada, etc.

## 🎬 Usamos a API do Omdb

Para enriquecer a coleta de informações, utilizamos a API pública do OMDb, disponível gratuitamente na web. Com ela, foi possível praticar o consumo de APIs e a manipulação de dados em formato JSON, tornando o projeto mais dinâmico e próximo de aplicações reais.
